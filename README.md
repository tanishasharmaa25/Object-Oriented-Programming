# Object-Oriented Programming

## 📌 Overview
This repository contains implementations of core Object-Oriented Programming (OOP) concepts using Python.  
The programs are designed as part of lab exercises and assignments, focusing on clean design, modularity, and practical application of OOP principles.

---

## 🎯 Objectives
- Understand and implement OOP concepts in Python
- Apply abstraction, encapsulation, inheritance, and polymorphism
- Design reusable and modular code
- Implement operator overloading and magic methods
- Solve mathematical problems using OOP design

---

## 🧠 Concepts Covered
- Classes and Objects  
- Constructors and Instance Variables  
- Getter and Setter Methods  
- Static and Class Methods  
- Magic Methods (`__init__`, `__str__`, `__repr__`, etc.)  
- Operator Overloading  
- Iterators (`__iter__`)  
- Callable Objects (`__call__`)  
- Exception Handling
