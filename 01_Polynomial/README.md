# Polynomial Class

## 📌 Overview
This module implements a **Polynomial class** in Python using Object-Oriented Programming principles.  
It supports various polynomial operations such as addition, subtraction, multiplication, evaluation, differentiation, and root finding.

---

## 🎯 Objectives
- Represent polynomials using lists
- Perform arithmetic operations on polynomials
- Apply operator overloading for intuitive usage
- Implement mathematical methods like derivative and root finding
- Demonstrate clean OOP design

---

## 🧠 Concepts Used
- Classes and Objects  
- Encapsulation  
- Magic Methods (`__init__`, `__str__`, `__repr__`)  
- Operator Overloading (`+`, `-`, `*`)  
- List manipulation  
- Exception Handling
