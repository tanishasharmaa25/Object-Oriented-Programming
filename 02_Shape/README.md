# Shape Class (Inheritance & Polymorphism)

## 📌 Overview
This module demonstrates Object-Oriented Programming concepts using a **Shape base class** and its derived classes.  
It showcases how inheritance and polymorphism can be used to design reusable and extensible code for geometric shapes.

---

## 🎯 Objectives
- Implement a base class and derived classes
- Demonstrate inheritance and method overriding
- Apply polymorphism for uniform interface
- Compute area and perimeter of different shapes

---

## 🧠 Concepts Used
- Classes and Objects  
- Inheritance  
- Polymorphism  
- Method Overriding  
- Encapsulation  
- Abstract Design
